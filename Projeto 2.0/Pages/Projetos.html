  • Projeto para solução da ODS 10.2

O projeto do banco de dados para consultas empresariais tem como objetivo democratizar o acesso aos dados das empresas aos clientes, fazendo com que o cliente tenha uma maior ciência das empresas nas quais nos relacionamos, isso na versão limitada. Tendo em vista a lucratividade, teremos uma versão exclusica onde será disponipilizados para captação de leads para empresas, onde vai ser montado um filtro de empresas especificas, a partir disso será gerada uma lista com apenas empresas selecionada, essa lista será disponibilizadas para que outras empresas possa entrar em contato oferecendo serviços
