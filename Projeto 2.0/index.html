Sobre mim

Estudante De Análise e Desenvolvimento para Sistemas

Principais objetivos durante o período no curso:

  • Análise de dados
  • Análise de banco in Cloud
  • Desenvolvimento de softwares
  • Desenvolvimento de Banco de Dados
  • Gerenciamento em AWS

Meu contato com a tecnologia da informação e computadores começou desde cedo. Desde muito novo, sempre tive uma afinidade especial com hardware. Com o tempo, meu interesse por infraestrutura de servidores e redes cresceu consideravelmente, enquanto o desenvolvimento de software sempre me pareceu menos atraente. A minha experiência no curso de Análise e Desenvolvimento de Sistemas tem sido uma oportunidade para sair da minha zona de conforto, e essa jornada tem se revelado um desafio muito gratificante.

Atualmente, um dos meus principais focos tem sido a área de Banco de Dados, que se tornou uma grande paixão. Essa área passou a ser o foco de alguns projetos. Meu contato com bancos de dados veio através de um emprego, onde tive a oportunidade de me familiarizar com o tema e criar afinidade. Isso me levou a integrar o uso de bancos de dados em boa parte dos projetos distintos que desenvolvo.

Meu maior desafio tem sido o gerenciamento do tempo, já que enfrento uma grande quantidade de demandas para resolver em pouco tempo. Isso acaba limitando meu tempo para estudos e o desenvolvimento de novos projetos. Após finalizar os projetos em que estou me dedicando, pretendo me empenhar na área de computação em nuvem, principalmente em AWS, que tem se expandido a cada dia mais no mercado.
